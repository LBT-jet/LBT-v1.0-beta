This version of LBT codes is used to simulate jet propagation in a static and uniform QCD medium, see [arXiv:2306.13742](https://arxiv.org/pdf/2306.13742)

How to run in the code folder:
1. Download the necessary rate tables from [figshare:tables](https://doi.org/10.6084/m9.figshare.33030944)
2. make clean
3. make
4. ./lbt

The analysis histogram is stored in LBTjet.root without particle output files.
